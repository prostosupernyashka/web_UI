# Интерактивный дашборд UI-компонентов

Кратко: SPA на чистом JS (ES6 modules). Виджеты: ToDo (локально), Quote (через API quotable), Advice (через API adviceslip). Проект демонстрирует ООП: UIComponent -> наследники.

## Запуск локально
1. Клонировать репозиторий.
2. Открыть `index.html` через локальный сервер (рекомендуется `live-server` или `python -m http.server`).

## Деплой на GitHub Pages
1. Создать репозиторий на GitHub.
2. Закоммитить файлы.
3. В настройках репозитория (Settings → Pages) выбрать ветку `main` и папку `/root` (или `gh-pages`).
4. Через несколько минут сайт будет доступен по адресу `https://<username>.github.io/<repo>/`.

## Использование
- Кнопки в шапке добавляют виджеты.
- Каждый виджет можно минимизировать и закрыть.
- Quote и Advice берут данные с внешних API.
